# julia2rust (WIP)

## Documentation

An inefficient one-filer Julia to Rust transpiler.

### Install

`git clone https://github.com/Jdf2023/julia2rust/tree/main`
`cd julia2rust`
`nimble build` for debug build
`nimble build -d:release` for release build
